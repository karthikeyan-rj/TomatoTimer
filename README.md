# TomatoTimer 🍅

A Pomodoro timer built with Python and Tkinter.

## How to Run

python main.py

Make sure `tomato.png` is in the same folder.

## How it Works
- 30 min work → 5 min break → repeat
- Long break after every 4 sessions
- Tracks sessions with ✔ marks

## Built With
- Python 3
- Tkinter
